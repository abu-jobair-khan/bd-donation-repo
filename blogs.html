Important FAQ

What is DOM

The Document Object Model is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document.

How do you select an element from the DOM?

Selecting elements from the DOM involves some selectors, these are getElementsByTagName,getElementsByClassName,getElemtById etc.

what is event delegation in the context of the dom and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future
